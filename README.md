# Wiki LLM Android

Локальный LLM-чат с офлайн-Википедией для Android. Скачивает GGUF-модели с Hugging Face и ZIM-файлы с Kiwix, отвечает на вопросы используя статьи из локальной Википедии как контекст (RAG).

## Статус: этап 1 / 7 — скелет

На этом этапе приложение содержит только навигацию между экранами. Модели и Википедия пока заглушки. Цель этапа — убедиться, что:
- GitHub Actions собирает APK без ошибок
- APK устанавливается на ваш телефон
- Приложение запускается и показывает 4 экрана: Главный, Модели, Википедия, Чат

## Как собрать APK

Ничего ставить на ПК не надо. Сборка идёт на GitHub Actions автоматически.

1. Сделайте push в ветку `main`
2. Зайдите во вкладку **Actions** репозитория — там пойдёт сборка (5–10 минут)
3. Когда сборка завершится, APK появится в двух местах:
   - В **Releases** репозитория (рекомендуется — открывается с телефона по ссылке)
   - В **Actions → последний запуск → Artifacts** (нужен логин в GitHub с телефона)

## Как установить на телефон

1. Откройте Releases репозитория с телефона
2. Скачайте `wiki-llm-NN-debug.apk`
3. Тапните по нему — Android спросит разрешение на установку из неизвестных источников
4. Разрешите для браузера/файлового менеджера и установите

## Технологический стек

- Kotlin + Jetpack Compose (UI)
- Material 3 + динамические цвета
- Navigation Compose
- Минимальная версия Android: API 31 (Android 12)
- Целевая: API 34 (Android 14)

На следующих этапах добавится:
- llama.cpp (C++ через NDK + JNI)
- libzim (C++ через NDK + JNI)
- Ktor / OkHttp для скачивания моделей и ZIM
- Room для кэша метаданных
